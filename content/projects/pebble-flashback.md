## Flashback for Pebble

[![](https://img.shields.io/badge/Github-pebble--flashback-blue?logo=github)](https://github.com/thementalgoose/pebble-flashback)

[![](https://img.shields.io/badge/Repebble_App_Store-Flashback_F1_Results-blue?color=orange)](https://apps.repebble.com/6929da3470731e00092debea)

A small pebble watchapp to view the flashback API data directly on your wrist!

- Supports viewing upcoming races
- Supports viewing race schedule details
- Supports viewing current drivers standings
- Supports viewing current teams standings
- Adds timeline pins for upcoming events
